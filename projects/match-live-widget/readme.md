# Виджет матча ЦСКА vs Зенит

## Описание
Интерактивный виджет для отображения информации о футбольном матче между ЦСКА и Зенитом в рамках ФОНБЕТ Кубка России. Виджет отображает основную информацию о матче, составы команд, события матча и статистику.

## Функционал
- Отображение информации о турнире и времени матча
- Отображение логотипов и названий команд
- Отображение счета матча
- Интерактивные вкладки с информацией:
  - Статистика матча (владение мячом, удары, угловые и т.д.)
  - События матча (голы, карточки, замены) на таймлайне
  - Составы команд с номерами игроков
- Адаптивный дизайн для различных устройств
- Автоматическая загрузка данных через GraphQL API Sports.ru

## Технологии
- HTML5
- CSS3 с использованием переменных для брендированных цветов
- Vanilla JavaScript
- GraphQL для получения данных матча
- Адаптивный и отзывчивый дизайн
- Поддержка различных состояний матча (не начался, в прямом эфире, завершен)
