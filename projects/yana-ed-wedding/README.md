# Свадебная страница Яны и Эдуарда

Полностью адаптивная свадебная лендинг-страница с 10 уникальными экранами, множественными декорациями из детских рисунков и компактным дизайном без отступов.

## Описание

Элегантная и полностью адаптивная свадебная страница для Яны и Эдуарда с датой торжества 30 августа 2025 года в стиле "пока родителей нет дома" с ностальгическими элементами 90-х.

## Последние обновления
- **Убраны подписи**: Удалены слова "невеста" и "жених" с первого экрана

- **Добавлено фото zal.jpg**: На экран с локацией добавлена фотография зала
- **Объединены экраны**: 8 и 9 экраны подарков объединены в один

- **Повышена видимость**: Декорации стали более заметными (opacity 0.6-0.8)
- **Лучшее распределение**: Декорации размещены по всем зонам экрана

    position: absolute;

    height: 120px;
    background-size: contain;
    background-repeat: no-repeat;
    opacity: 0.8;
    z-index: 0;

/* Дополнительные декорации через .decoration-1, .decoration-2, .decoration-3 */
