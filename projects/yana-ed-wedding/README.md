# Свадебная страница Яны и Эдуарда

Компактная адаптивная свадебная лендинг-страница с плавным скроллом, без декораций, с новыми фонами и минимальными отступами.

## Описание

Элегантная и полностью адаптивная свадебная страница для Яны и Эдуарда с датой торжества 30 августа 2025 года в стиле "пока родителей нет дома" с ностальгическими элементами 90-х.

## Последние обновления
- **Компактность на десктопе**: Уменьшены отступы секций с 80px до 40px для более плотного контента
- **Убрано видео**: Удалено mp4 видео love из-за проблем с воспроизведением  
- **Новый фон Timeline**: Заменен на panel.jpg с приглушением (rgba 0.7) для читаемости текста
- **Новый фон POV**: Заменен на parents.jpg с приглушением (rgba 0.7) для читаемости текста
- **Убраны лишние отступы**: Минимизированы все отступы между секциями и внутри них
- **Улучшенная читаемость**: Усилен text-shadow для текста на фоновых изображениях

## Техническая реализация

### Компактные отступы
```css
section {
    padding: 40px 0; /* Было 80px */
}

.intro-section,
.date-section {
    padding: 60px 0; /* Только для главных секций */
}
```

### Новые фоны с приглушением
```css
.timeline-section {
    background-image: url('./images/panel.jpg');
}

.concept-section {
    background-image: url('./images/parents.jpg');
}

.timeline-section::before,
.concept-section::before {
    background: rgba(255, 255, 255, 0.7); /* Усиленное приглушение */
}
```

### Улучшенная читаемость текста
```css
.timeline-desc,
.concept-text p {
    text-shadow: 1px 1px 3px rgba(255, 255, 255, 0.9); /* Усиленный shadow */
    font-weight: 600;
}
```

### Убраны лишние отступы
```css
.section-title {
    margin-bottom: 1.5rem; /* Было 2rem */
}

.reality-text {
    margin: 20px 0 0 0; /* Убран большой нижний отступ */
}

.gifts-alt-content {
    gap: 30px; /* Было 40px */
}

.slider-group {
    margin-bottom: 15px; /* Было 20px */
}
```
