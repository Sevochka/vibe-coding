# Свадебная страница Яны и Эдуарда

Полностью адаптивная свадебная лендинг-страница с 10 уникальными экранами, симметричными декорациями из детских рисунков и улучшенным дизайном.

## Описание

Элегантная и полностью адаптивная свадебная страница для Яны и Эдуарда с датой торжества 30 августа 2025 года в стиле "пока родителей нет дома" с ностальгическими элементами 90-х.

## Последние обновления (финальная версия)
- **Симметричные декорации**: Детские рисунки размещены аккуратно вокруг контента, не наезжая на текст и слайдеры
- **Увеличено количество декораций**: До 6 декораций на экран (кроме экрана даты)
- **Убраны декорации с экрана даты**: Экран "Встречаемся" теперь только с фоном, текстом и фото
- **Увеличено фото love.jpg**: Размер увеличен до 400px для лучшей видимости
- **Улучшена видимость текста**: Подзаголовок на белой подложке для читаемости на фоне
- **Выровнены фотографии локации**: zal.jpg и Mondrian.jpg приведены к одинаковым пропорциям (350x250px)
- **Адаптивный дизайн**: Все декорации корректно масштабируются на мобильных устройствах

## Особенности
- **Симметричное размещение**: 6 декораций на каждом экране (кроме даты) в углах и по бокам
- **Чистый экран даты**: Только фон fon.jpg, текст с подложкой и увеличенное фото love.jpg
- **Единообразные фото**: Все изображения локации имеют одинаковые размеры и пропорции
- **Не мешают контенту**: Декорации размещены так, чтобы не перекрывать текст и интерактивные элементы
- **Полная адаптивность**: Корректное отображение от 480px до десктопа

## Структура сайта (10 экранов)

- **Не мешают тексту**: Размещены аккуратно вокруг контента
- **Адаптивные**: Уменьшаются на мобильных устройствах

### 2. Экран - Дата встречи ⭐ ОБНОВЛЕН
**Фоновое изображение fon.jpg** БЕЗ декораций
- **Чистый дизайн**: Убраны все детские рисунки для акцента на содержании
- **Улучшенная читаемость**: Подзаголовок на белой полупрозрачной подложке

- **Единые пропорции**: object-fit: cover для красивого отображения

### 4. Экран - Тайминг

## Техническая реализация декораций

### Симметричная система (6 декораций на экран)
```css
/* Основные декорации в углах */
section::before,
section::after {
    content: '';
    position: absolute;
    width: 80px;
    height: 80px;
    opacity: 0.6;
    /* Верхние углы */
}

/* Дополнительные декорации */
.section .decoration-1,
.section .decoration-2 {
    /* Нижние углы */
    width: 70px;
    height: 70px;
    opacity: 0.5;
}

.section .decoration-3,
.section .decoration-4 {
    /* По бокам */
    width: 60px;
    height: 60px;
    opacity: 0.4;
}
```

### Чистый экран даты
```css
.date-section::before,
.date-section::after,
.date-section .decoration-1,
.date-section .decoration-2,
.date-section .decoration-3 {
    display: none; /* Убираем все декорации */
}

.date-subtitle {
    background: rgba(255, 255, 255, 0.8);
    padding: 10px 20px;
    border-radius: 10px;
    display: inline-block;
}
```

### Выровненные фотографии локации
```css
.zal-photo,
.mondrian-photo {
    width: 100%;
    max-width: 350px;
    height: 250px;
    object-fit: cover;
    object-position: center;
}
```

## Адаптивные брейкпоинты

### Десктоп (по умолчанию)
- **6 декораций на экран**: Полноразмерные (80px, 70px, 60px)
- **Симметричное размещение**: В углах и по бокам
- **Не мешают контенту**: Аккуратное позиционирование

### Планшеты (1024px и меньше)
- **Уменьшенные декорации**: 60px, 50px, 40px
- **Сохранение симметрии**: Пропорциональное уменьшение
- **Фото локации**: 300x200px

### Мобильные (768px и меньше)
- **Компактные декорации**: 45px, 35px, 30px
- **Читаемость**: Приоритет контенту над декорациями
- **Фото локации**: 280x180px

### Маленькие экраны (480px и меньше)
- **Минимальные декорации**: 30px, 25px, 20px
- **Сохранение функциональности**: Все элементы доступны
- **Фото локации**: 250x160px
