# Свадебная страница Яны и Эдуарда

Полностью адаптивная свадебная лендинг-страница с 11 уникальными экранами, множественными декорациями, компактным дизайном без отступов и улучшенным layout.

## Описание

Элегантная и полностью адаптивная свадебная страница для Яны и Эдуарда с датой торжества 30 августа 2025 года в стиле "пока родителей нет дома" с ностальгическими элементами 90-х.

## Особенности
- **Полностью убранные отступы**: Все padding между экранами убраны для компактного отображения
- **Цвет заголовка как сердечко**: Главный заголовок окрашен в цвет сердечка с изображения fon
- **Подписи к фотографиям**: "невеста" под девочкой и "жених" под мальчиком на первом экране
- **Фоновые изображения**: Экран "Встречаемся" с фоном fon.jpg
- **Дополнительные изображения**: love.jpg под датой, Mondrian.jpg в локации
- **Вертикальные слайдеры**: Расположены друг над другом для удобного просмотра
- **Уменьшенный текст денег**: В два раза меньше с сокращенными отступами
- **Множественные декорации**: Каждый экран содержит 4-5 детских рисунков с повышенной видимостью
- **Широкий layout**: Увеличена ширина контейнера до 900px
- **Адаптивный дизайн**: Корректное отображение на всех устройствах
