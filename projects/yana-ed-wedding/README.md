# Свадебная страница Яны и Эдуарда

Полностью адаптивная свадебная лендинг-страница с 11 уникальными экранами, оптимизированная для всех устройств с использованием рукописного шрифта Bad Script.

## Описание

Элегантная и полностью адаптивная свадебная страница для Яны и Эдуарда с датой торжества 30 августа 2025 года. Дизайн построен на концепции "пока родителей нет дома" с рукописной типографикой и полноэкранными секциями.

## Основные изменения в последней версии

### Новая типографика
- **Bad Script**: Единый рукописный шрифт от Google Fonts для всего текста
- **Читаемость**: Увеличенные размеры шрифтов для комфортного чтения
- **Аутентичность**: Рукописный стиль придает личный характер

### Полная адаптивность
- **Полноэкранные секции**: Каждая секция занимает минимум 100vh на десктопе
- **Flex-центрирование**: Контент центрируется по вертикали и горизонтали
- **Четкие брейкпоинты**: 
  - Десктоп: полноэкранные секции с крупными элементами
  - Планшет (1024px): адаптация размеров
  - Мобильный (768px): перестройка layout в вертикальные блоки
  - Маленький мобильный (480px): компактные размеры
- **Отзывчивые изображения**: Все фото масштабируются правильно

### Улучшенный дизайн
- **Контейнеры**: Все блоки имеют белые полупрозрачные подложки
- **Центрирование**: Flex-системы для идеального выравнивания
- **Пропорции**: Правильные отступы и размеры для всех экранов

## Структура сайта

### 1. Экран - Знакомство
**Полноэкранная секция** с горизонтальным layout
- **Фото**: 300px ширина на десктопе, адаптируется на мобильных
- **Текст**: Рукописный шрифт Bad Script 1.3rem
- **Адаптивность**: Вертикальный стек на мобильных устройствах

**Полноэкранная секция** с центрированным контентом

- **Обратный отсчет**: Крупные блоки с автообновлением
- **Адаптивность**: Счетчик в одну линию на десктопе, перестраивается на мобильных

### 3. Экран - Локация
**Полноэкранная секция** с информацией о месте

- **Примечание**: Курсивное оформление

**Полноэкранная секция** с timeline
- **Временные блоки**: Крупные блоки 180px минимальная ширина
- **Описания**: Bad Script 1.3rem
- **Адаптивность**: Горизонтальные блоки становятся вертикальными

### 5. Экран - Концепция вечера
**Полноэкранная секция** с описанием концепции
- **Подзаголовок**: Bad Script 2.2rem "POV: пока родителей нет дома"

### 6. Экран - Дресс-код
**Полноэкранная секция** с цветовой палитрой
- **Цветовые блоки**: Увеличенные до 60px на десктопе
- **Описания**: Четкие подписи под каждым цветом
- **Адаптивность**: Блоки перестраиваются в зависимости от ширины экрана

### 7. Экран - Ожидания
**Полноэкранная секция** с каруселью
- **Карусель**: Увеличенная высота изображений (350px)

### 8. Экран - Творческие подарки

- **Текст**: Увеличенный размер Bad Script для лучшей читаемости

### 9. Экран - Денежные подарки

### 10. Экран - Альтернатива + Контакты
**Полноэкранная секция** с двумя блоками
- **Контакты организатора**: Выделенный блок с контактной информацией

**Полноэкранная секция** с финальным обращением
- **Фото пары**: Два фото справа в колонку
- **Подпись**: Bad Script розовым цветом
- **Адаптивность**: Фото переходят в горизонтальную линию на планшетах

## Техническая реализация

### Полная адаптивность
- **Mobile First**: Подход с постепенным улучшением для больших экранов
- **Flexbox**: Для гибкого расположения и центрирования элементов
- **Viewport Units**: Использование 100vh для полноэкранных секций
- **Отзывчивые изображения**: object-fit: cover для правильного масштабирования

### Типографика Bad Script
- **Google Fonts**: Импорт рукописного шрифта Bad Script
- **Единообразие**: Один шрифт для всего контента
- **Масштабируемость**: Размеры адаптируются под размер экрана
- **Читаемость**: Оптимальные размеры для каждого устройства

### Производительность
- **Единый шрифт**: Быстрая загрузка одного font-family
- **Оптимизированные медиа-запросы**: Четкие брейкпоинты
- **CSS Grid и Flexbox**: Современные layout-системы
- **Минимум JavaScript**: Только необходимый функционал

### Интерактивность
- **Карусель**: Автоматическая смена с ручным управлением
- **Обратный отсчет**: Живой таймер до события
- **Кнопка "наверх"**: Для удобной навигации по длинной странице
- **Hover-эффекты**: На цветовых блоках и кнопках

## Адаптивные брейкпоинты

### Десктоп (по умолчанию)
- **Полноэкранные секции**: min-height: 100vh с flex-центрированием
- **Крупные элементы**: Оптимальные размеры для больших экранов
- **Горизонтальные layout**: Фото рядом с текстом

### Большие планшеты (1024px и меньше)
- **Адаптация контейнера**: Полная ширина с отступами 30px
- **Уменьшение размеров**: Пропорциональное масштабирование элементов

### Планшеты (768px и меньше)
- **Вертикальный layout**: Все горизонтальные блоки становятся вертикальными
- **Адаптация timeline**: Временные блоки центрируются
- **Фото пары**: Горизонтальное расположение для экономии места

### Мобильные (480px и меньше)
- **Компактные размеры**: Уменьшенные отступы и размеры элементов
- **Вертикальное расположение**: Всё в одну колонку
- **Оптимизированная типографика**: Читаемые размеры для маленьких экранов

## Цветовая схема

### Основные цвета
- **Текст**: #2C2C2C (темно-серый)
- **Серый текст**: #666666 (для второстепенной информации)
- **Фон**: #FFFFFF (белый)
- **Акцент**: #FF6B9D (розовый для подписи и элементов)

### Дресс-код цвета
- **Акцентные**: Розовый (#FF6B9D), синий (#4A90E2), желтый (#FFD93D), зеленый (#4CAF50)
- **Базовые**: Серый, бежевый, черный, коричневый
