# Свадебная страница Яны и Эдуарда

Полностью адаптивная свадебная лендинг-страница с 11 уникальными экранами, оптимизированная для всех устройств с новой типографикой Pangolin и Poiret One.

## Описание

Элегантная и полностью адаптивная свадебная страница для Яны и Эдуарда с датой торжества 30 августа 2025 года. Дизайн построен на концепции "пока родителей нет дома" с сочетанием рукописной и современной типографики в полноэкранных секциях.

## Последние изменения

### Новая типографическая система
- **Pangolin**: Рукописный шрифт Google Fonts для всех заголовков
- **Poiret One**: Современный sans-serif для основного текста
- **Улучшенная читаемость**: Оптимальные размеры для каждого устройства
- **Гармоничное сочетание**: Рукописные заголовки + чистый основной текст

### Структурные изменения
- **Убран обратный отсчет**: Удалена функциональность таймера со второго экрана
- **Новый заголовок**: "Свадьба Яны и Эда" над фотографией на первом экране
- **Жирная дата**: "30 августа 2025 года" выделена жирным шрифтом заголовков
- **Заголовок "Тайминг"**: Добавлен в экран с расписанием праздника
- **Переструктурированы контакты**: "Кому задать?" перенесено под "Остались вопросы."

### Полная адаптивность
- **Полноэкранные секции**: Каждая секция занимает минимум 100vh на десктопе
- **Flex-центрирование**: Контент центрируется по вертикали и горизонтали
- **Четкие брейкпоинты**: 
  - Десктоп: полноэкранные секции с крупными элементами
  - Планшет (1024px): адаптация размеров
  - Мобильный (768px): перестройка layout в вертикальные блоки
  - Маленький мобильный (480px): компактные размеры
- **Отзывчивые изображения**: Все фото масштабируются правильно

## Техническая реализация

### Новая типографическая система
- **Google Fonts**: Импорт Pangolin и Poiret One
- **Иерархия**: Четкое разделение заголовков и основного текста
- **Читаемость**: Оптимальные размеры для каждого экрана
- **Производительность**: Быстрая загрузка двух шрифтов

### Полная адаптивность
- **Mobile First**: Подход с постепенным улучшением для больших экранов
- **Flexbox**: Для гибкого расположения и центрирования элементов
- **Viewport Units**: Использование 100vh для полноэкранных секций
- **Отзывчивые изображения**: object-fit: cover для правильного масштабирования

### Интерактивность без счетчика
- **Двойные слайдеры**: Два независимых слайдера с разными интервалами автопрокрутки
- **Индивидуальное управление**: Каждый слайдер имеет свои кнопки навигации
- **Пауза при наведении**: Автопрокрутка останавливается при hover на каждом слайдере отдельно
- **Кнопка "наверх"**: Для удобной навигации
- **Hover-эффекты**: На цветовых блоках, фотографиях
- **Плавная прокрутка**: Между секциями
- **Анимации появления**: При скролле

### Оптимизация производительности
- **Упрощенный JavaScript**: Убраны функции обратного отсчета
- **Минимум зависимостей**: Только необходимый функционал
- **CSS Grid и Flexbox**: Современные layout-системы
- **Lazy loading эффектов**: Постепенная инициализация

## Адаптивные брейкпоинты

### Десктоп (по умолчанию)
- **Полноэкранные секции**: min-height: 100vh с flex-центрированием
- **Крупная типографика**: Pangolin заголовки до 3.5rem
- **Горизонтальные layout**: Фото рядом с текстом

### Большие планшеты (1024px и меньше)
- **Адаптация контейнера**: Полная ширина с отступами 30px
- **Уменьшение заголовков**: Пропорциональное масштабирование

### Планшеты (768px и меньше)
- **Вертикальный layout**: Все горизонтальные блоки становятся вертикальными
- **Адаптация timeline**: Временные блоки центрируются
- **Уменьшенные заголовки**: Pangolin заголовки до 2.8rem

### Мобильные (480px и меньше)
- **Компактная типографика**: Заголовки от 1.8rem
- **Вертикальное расположение**: Всё в одну колонку
- **Оптимизированные размеры**: Комфортное чтение на малых экранах

## Особенности контента

### Новая типографическая гармония
- **Pangolin**: Создает теплую, личную атмосферу в заголовках
- **Poiret One**: Обеспечивает отличную читаемость основного текста
- **Контраст**: Рукописное + геометричное для современного вида

### Упрощенная структура
- **Без таймера**: Фокус на информации, а не на отсчете времени
- **Четкая иерархия**: Каждый заголовок имеет свое назначение
- **Логичный порядок**: От общего к частному, от информации к контактам

### Практичность и эстетика
- **Вся необходимая информация**: Время, место, дресс-код, контакты
- **Четкие контакты**: Понятно, к кому обращаться с вопросами
- **Мобильная готовность**: Идеальная работа на всех устройствах
- **Визуальная привлекательность**: Сочетание функциональности и красоты

Сайт создает атмосферу личного приглашения с современной типографикой, полностью адаптированным для комфортного просмотра на любых устройствах - от больших мониторов до смартфонов.
