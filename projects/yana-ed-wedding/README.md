# Свадебная страница Яны и Эдуарда

Полностью адаптивная свадебная лендинг-страница с 11 уникальными экранами, оптимизированная для всех устройств с использованием современной типографики.

## Описание

Элегантная и полностью адаптивная свадебная страница для Яны и Эдуарда с датой торжества 30 августа 2025 года. Дизайн построен на концепции "пока родителей нет дома" с современной типографикой и полноэкранными секциями.

## Последние изменения

### Новая типографика
- **Pangolin**: Дружелюбный рукописный шрифт для всех заголовков
- **Poiret One**: Элегантный геометрический шрифт для основного текста
- **Контраст**: Сочетание создает идеальный баланс между игривостью и элегантностью

### Структурные изменения
- **Убран обратный отсчет**: Упрощена секция с датой для лучшего восприятия
- **Добавлен главный заголовок**: "Свадьба Яны и Эда" над основным фото
- **Жирная дата**: "30 августа 2025 года" выделена шрифтом заголовков
- **Заголовок "Тайминг"**: Добавлен в секцию с расписанием
- **Разделение контактов**: "Остались вопросы?" и "Кому их задать?" теперь отдельные заголовки

### Полная адаптивность
- **Полноэкранные секции**: Каждая секция занимает минимум 100vh на десктопе
- **Flex-центрирование**: Контент центрируется по вертикали и горизонтали
- **Четкие брейкпоинты**: 
  - Десктоп: полноэкранные секции с крупными элементами
  - Планшет (1024px): адаптация размеров
  - Мобильный (768px): перестройка layout в вертикальные блоки
  - Маленький мобильный (480px): компактные размеры

## Техническая реализация

### Современная типографика
- **Google Fonts**: Импорт Pangolin и Poiret One
- **Иерархия**: Четкое разделение заголовков и основного текста
- **Читаемость**: Оптимальные размеры для каждого устройства
- **Контраст**: Игривые заголовки + элегантный основной текст

### Полная адаптивность
- **Mobile First**: Подход с постепенным улучшением для больших экранов
- **Flexbox**: Для гибкого расположения и центрирования элементов
- **Viewport Units**: Использование 100vh для полноэкранных секций
- **Отзывчивые изображения**: object-fit: cover для правильного масштабирования

### Упрощенный функционал
- **Убран обратный отсчет**: Более чистый дизайн без лишних элементов
- **Улучшенная структура**: Логичное разделение заголовков и подзаголовков
- **Оптимизированный JavaScript**: Только необходимые функции

### Интерактивность
- **Карусель**: Автоматическая смена с ручным управлением
- **Кнопка "наверх"**: Для удобной навигации по длинной странице
- **Hover-эффекты**: На цветовых блоках и кнопках
- **Анимации прокрутки**: Плавное появление секций
