# Свадебная страница Яны и Эдуарда

Единая адаптивная свадебная лендинг-страница с плавным скроллом, без декораций, с видео и современным минималистичным дизайном.

## Описание

Элегантная и полностью адаптивная свадебная страница для Яны и Эдуарда с датой торжества 30 августа 2025 года в стиле "пока родителей нет дома" с ностальгическими элементами 90-х.

## Кардинальные обновления
- **Убраны все декорации**: Полностью удалены детские рисунки с сайта
- **Единая страница**: Не разделена на экраны, один скролл с плавной анимацией (scroll-behavior: smooth)
- **Нежно-розовый фон**: Второй экран с датой имеет мягкий розовый фон (#FFE4E6)
- **Увеличенное фото love**: Максимальная ширина 500px вместо 300px
- **Прозрачные фоны**: Timeline и Concept без подложек, текст читается благодаря text-shadow
- **Видео love**: Добавлено автоматически воспроизводящееся видео в красивой рамке перед финальным текстом
- **Горизонтальные фото**: Финальные фотографии увеличены до 300x200px в горизонтальном формате

## Техническая реализация

### Плавный скролл
```css
html {
    scroll-behavior: smooth;
}
```

### Единая страница
```css
section {
    padding: 80px 0; /* Вместо min-height: 100vh */
    /* Секции теперь не занимают полную высоту экрана */
}
```

### Убраны все декорации
```css
section::before,
section::after,
.decoration-1,
.decoration-2,
.decoration-3,
.decoration-4,
.decoration-5,
.decoration-6 {
    display: none !important;
}
```

### Нежно-розовый фон
```css
.date-section {
    background: var(--soft-pink-bg); /* #FFE4E6 */
}
```

### Увеличенное фото love
```css
.love-photo {
    max-width: 500px; /* Было 300px */
}
```

### Прозрачные фоны с читаемым текстом
```css
.timeline-section::before,
.concept-section::before {
    background: rgba(255, 255, 255, 0.3);
}

.timeline-desc,
.concept-text p {
    text-shadow: 1px 1px 2px rgba(255, 255, 255, 0.8);
    font-weight: 600;
}
```

### Видео love
```css
.love-video-container {
    border: 3px solid var(--primary-pink);
    background: linear-gradient(45deg, #FFE4E6, #FFF0F1);
    border-radius: var(--border-radius);
}

.love-video {
    max-width: 500px;
    border-radius: var(--border-radius);
}
```

### Горизонтальные финальные фото
```css

    object-fit: cover;

```
