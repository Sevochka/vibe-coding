# Свадебная страница Яны и Эдуарда

Полностью адаптивная свадебная лендинг-страница с 11 уникальными экранами в пастельных тонах с декоративными элементами и рамками для текстов и фотографий.

## Описание

Элегантная и полностью адаптивная свадебная страница для Яны и Эдуарда с датой торжества 30 августа 2025 года. Дизайн построен на концепции "пока родителей нет дома" с современной типографикой, пастельными розово-бежевыми цветами и декоративными элементами.

## Техническая реализация

### Пастельный дизайн
- **Градиентный фон**: От пастельно-розового к пастельно-бежевому
- **Прозрачные оверлеи**: rgba для секций с разными оттенками
- **Декоративные элементы**: Фиксированные фоновые изображения
- **Мягкие тени**: Розовые тени вместо серых

### Рамки и декор
- **Текстовые рамки**: background-image с table.jpg для всех текстовых блоков
- **Фоторамки**: ramka.jpg для всех изображений
- **Адаптивные рамки**: Изменение padding в зависимости от экрана
- **Центрирование**: Flex для правильного позиционирования контента

### Декоративные элементы
- **Фоновые рисунки**: risunki.jpg и risunki2.jpg как ::before псевдоэлементы
- **Настольные игры**: nastolki.jpg в секции подарков через ::before и ::after
- **Z-index**: Правильное наслоение декоративных элементов
- **Opacity**: Полупрозрачность для ненавязчивости

### Адаптивность
- **Размеры рамок**: Адаптивные padding для разных экранов
- **Карусель**: Специальные стили для фото в рамках
- **Центрирование**: white-space: normal на мобильных
- **Декор**: Скрытие или изменение размеров декоративных элементов

### Цветовая схема

#### Пастельные цвета
- **Пастельно-розовый**: #FFE4E1 (основной фон)
- **Пастельно-бежевый**: #F5F5DC (альтернативный фон)
- **Прозрачный розовый**: rgba(255, 182, 193, 0.3)
- **Прозрачный бежевый**: rgba(245, 245, 220, 0.5)

#### Акценты
- **Розовый**: #FFB6C1 (кнопки, выделения)
- **Текст**: #2C2C2C (основной)
- **Серый текст**: #666666 (второстепенный)
- **Тени**: rgba(255, 182, 193, 0.3)

#### Стилизованные рамки
- **table.jpg**: Для всех текстовых блоков
- **ramka.jpg**: Для всех фотографий
- **Адаптивность**: Изменение размеров рамок на разных экранах
