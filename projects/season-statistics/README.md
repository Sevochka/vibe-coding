# Статистика сезона

Виджет для отображения статистики футбольных сезонов различных турниров. Виджет позволяет выбрать турнир, сезон и просмотреть основные статистические показатели команд и игроков.

## Описание

Проект представляет собой интерактивный виджет, который загружает и обрабатывает статистические данные матчей из JSON-файла. Пользователь может выбрать интересующий его турнир и сезон, после чего виджет отображает основные статистические показатели в удобном формате.

## Функциональность

- Выбор турнира из выпадающего списка
- Выбор сезона для выбранного турнира
- Просмотр статистики команд:
  - Лучшие по забитым голам
  - Лучшие по владению мячом
  - Лучшие по точности передач
  - Лучшие по количеству ударов
- Просмотр статистики игроков:
  - Лучшие бомбардиры
  - Лучшие ассистенты
  - Лучшие по системе гол+пас
  - Лидеры по желтым карточкам
- Адаптивный дизайн, соответствующий стилю Sports.ru

Виджет автоматически загружает данные из JSON-файла, обрабатывает их и представляет в удобном для пользователя виде с возможностью переключения между разными категориями статистики.
